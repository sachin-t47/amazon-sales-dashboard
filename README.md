# 🛒 Amazon Sales Analytics Dashboard

## 📌 Project Overview
End-to-end data analytics project analyzing 50,000+ Amazon sales transactions
to uncover revenue trends, category performance, and payment patterns.

## 🛠️ Tech Stack
- 🐍 **Python** — Data cleaning & EDA (Pandas, Matplotlib, Seaborn)
- 🗄️ **SQL** — Business analysis queries (SQLite)
- 📊 **Power BI** — Interactive dashboard

## 🔍 Key Insights
- 📦 Total Revenue: **₹32.87M** across 50K orders
- ⭐ Average Rating: **3.0** across all categories
- 💳 Payment Methods equally distributed (~20% each)
- 📈 Revenue trend shows seasonality in 2022-2023
